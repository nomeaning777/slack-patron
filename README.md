# Slack patron

|Name    | Example   |
|--------|-----------|
|`MONGO_DB_URL`| `mongodb://127.0.0.1/hogehoge` |

|`ENABLE_DIRECT_MESSAGE`| `1` or `0` |
|`AUTO_JOIN`| `1` or `0` |
